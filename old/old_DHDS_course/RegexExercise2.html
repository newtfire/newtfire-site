Your challenge is to up-convert to XML the complete plain text file of A VOYAGE round the WORLD by Georg Forster. This file is quite large, so autotagging using Regular Expressions (regex) is really the only option we have to make this text into an XML document. Begin by downlading the text file and opening it in <oXygen/>. Use the Find/Replace window to autotag the document, and review our Guide to Autotagging with Regular Expressions as you work. Your XML markup should accomplish the following:

  1. Indicate the structure of the file by marking Book divisions, chapter divisions, and paragraphs.
  2. Tag the dates, at least the dates that sitting in square brackets. Ideally, you should remove any pseudo-markup around them.

Your complete text should look like our model XML file, only you could go one better by removing the pseudo-markup (the brackets) around the dates. Can you locate and tag more dates than those in the square brackets?

  • Record each step of your process carefully, in a separate document (ideally a plain text file), since this is the file we will be grading. These will include global Find-and-Replace operations or Regular Expressions in oXygen (using Ctrl-F on Windows or command-F on Mac). Your goal is to produce an XML document like our model, but even if you have have trouble, what's most important is that you document the steps you took.
  • Upload two files on Courseweb for this exercise:
    1. the document in which you recorded your steps (either a text file or a Word document, whichever is convenient),
    2. your end result XML file.
